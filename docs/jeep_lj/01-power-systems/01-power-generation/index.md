# 1.1 Power Generation & Storage {#power-generation}

## System Overview

Dual Odyssey PC1500 batteries (850 CCA, 68 Ah each) with RedArc BCDC Alpha 25 isolation/charging and 200A alternator. Starter battery powers critical engine systems, aux battery powers accessories. Batteries operate independently when isolated, aux battery can assist starting via BCDC jump start mode. 80W hood solar panel maintains aux battery.
