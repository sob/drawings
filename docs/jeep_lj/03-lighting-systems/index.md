Street-legal, DOT-required, and interior vehicle lighting systems.

## Contents

- **[3.1 Lighting Overview][vehicle-lighting-overview]** - System architecture, controller assignments, lighting organization
- **[3.2 Headlights][headlights]** - Baja Designs LP6 DOT headlights with Command Touch CT4 control
- **[3.3 Turn Signals][turn-signals]** - DOT turn signals with GPS auto-cancel, diode isolation from brake circuit

## System Architecture

- Command Touch CT4: Headlights, turn signals
- PMU: DRL, brake, reverse, marker lights
- SwitchPros: Dome lights (with door trigger)
- DOT-compliant exterior, integrated with factory controls
