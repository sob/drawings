## Wiring

All speakers use JL Audio XM-WHTMFC integrated cable:

| Wire          | Gauge      | Function     |
| :------------ | :--------- | :----------- |
| Speaker leads | 16 AWG OFC | Audio signal |
| LED leads     | 20 AWG OFC | RGB + common |

Single cable run per speaker carries both audio and LED control.
