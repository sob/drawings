## System Overview

**Recovery Equipment:**
- Warn Zeon 10-S winch (10,000 lb capacity)
- SafetyHub inline fuse/contactor system
- Dash 3-position control switch

**Air System:**
- ARB Twin Compressor (brushless, 42A draw)
- 2-gallon air tank (135-150 PSI)
- ARB front/rear air lockers
- Automatic pressure management via SwitchPros
